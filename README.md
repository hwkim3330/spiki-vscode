# Spiki Tamagotchi

귀여운 스피키와 함께 코딩하세요! 코딩할수록 스피키가 행복해져요.

## Features

- 사이드바에서 스피키를 돌볼 수 있어요
- 코딩하면 스피키가 행복해져요
- 밥주기, 놀아주기, 쓰다듬기, 재우기 기능
- 레벨업 시스템
- 상태바에서 스피키 상태 확인

## Commands

- `Spiki: Show Spiki` - 스피키 패널 열기
- `Spiki: Feed Spiki` - 밥주기
- `Spiki: Play with Spiki` - 놀아주기
- `Spiki: Pet Spiki` - 쓰다듬기

## Settings

- `spiki.showStatusBar` - 상태바 표시 여부
- `spiki.autoFeed` - 코딩 활동 자동 보상
